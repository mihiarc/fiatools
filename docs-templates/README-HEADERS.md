## Badge Style Guide

Use shields.io badges with consistent styling:

```
https://img.shields.io/badge/[LABEL]-[MESSAGE]-[COLOR]?style=flat
```

Recommended badge order:
1. Documentation link
2. License
3. Python version
4. Build status (if applicable)
5. Coverage (if applicable)
